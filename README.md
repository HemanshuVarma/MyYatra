# MyYatra - An Tour Guide App

This App suggests **Top 5** _Places, Hotels, Food and Adventure Places_ in India.

### Concepts used:

* CardView
* Fragments
* Custom Class
* ListView
* TabLayout
* ViewPager
* ViewHolder
* Adapter
